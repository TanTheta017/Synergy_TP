# Task 1 : GitHub , Virtual Environment and Linux Basics

## Description:
Demonstartes basic GitHub usage,Python virtual environment setup, dependency management and linux commands_line roots

## Setup Instructions

### Clone the Repository
```bash
git clone https://github.com/TanTheta017/Synergy_TP.git
cd Synergy_TP 
```

### Create VE
```bash
python -m venv venv
```

### Activating VE
```bash
venv\Scripts\activate
```

### Install Requirements
```bash
pip install -r task_1/requiremnets.txt
```

### Run the python Script
```bash
python task_1/src.hello.py
```
